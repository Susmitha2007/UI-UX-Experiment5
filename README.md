## Aim:

To present two different UX (User Experience) project scenarios to understand and apply UX design principles in real-world contexts using suitable design tools.

## Algorithm:

1. Onboarding & Signup (common)

Purpose: register user, collect minimal profile, set initial preferences/goals.
Steps:

Display welcome / benefits screen.

Offer signup options: Email, Google/Facebook.

If social signup → fetch basic profile (name, email).

If email signup → validate email & password.

Ask minimal profile questions (age, weight, goals) — optional skip.
